## About

This repository serves as a companion resource for the Angry AI with Ty YouTube channel. Here you'll find:

- **Guides & Tutorials** — Step-by-step documentation referenced in videos
- **Scripts** — Code and automation scripts featured on the channel
- **Benchmarks** — Stored outputs from local model and hardware evaluation runs
- **Resources** — Screenshots, diagrams, and other supporting materials
